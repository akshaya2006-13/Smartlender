System Architecture & Pipeline

About SMART LENDER Project

The SMART LENDER system represents an intelligent machine learning underwriting tool designed to determine borrower eligibility and compute credit risk scores.

By processing customer demographic and financial details, the platform determines pre-approvals instantly while checking parameters against a collection of eight optimized classifiers. The system uses a modular pipeline to guarantee that input processing matches training standards, eliminating data leakage and ensuring reproducible predictions.

Model Training Pipeline Workflow

  1. Missing Value Imputation: Imputes missing numerical columns with the training median and missing categorical columns with the mode.
  2. Outlier Management: Log transformation ($log(x+1)$) is applied to skewed features (Applicant Income, Co-applicant Income, and Loan Amount) to normalize distributions.
  3. Feature Encoding: Binary categoricals are label-encoded, while multi-class variables (Property Area) are hot-encoded.
  4. Standard Scaling: Features are scaled using standard normalization before being passed to classifiers.
  5. Cross-Validation Comparison: 8 classification models are trained and compared across accuracy, precision, recall, and F1 scores. The best model is selected automatically.
